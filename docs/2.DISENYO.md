__GLOSARIO__

iot: internet of thing. Objeto con conexión a internet que ofrece datos publicos (nombre y brand) y datos privados (acciones y propiedades)

iot_emulator: recurso web donde configurar emulaciones de iot

dueño: persona que posee iots, conoce las credenciales para acceder a zona privada de iot. Además posee cuenta de facebook y red de amigos dentro de esta red social

amigo: persona conectada como amigo en facebook

sac: social access controller. Aplicación acoplada a facebook donde un dueño puede compartir cierta acción con un amigo.
